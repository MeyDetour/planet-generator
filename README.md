# APIs utilisées

### **Nom aléatoire de planète**  
L'API [cjglitter](https://www.cjglitter.com/rand_name/api) permet d'obtenir un nom aléatoire pour chaque planète générée.

### **Accès en développement local**  
En local, l'accès à l'API cjglitter est bloqué par CORS. Pour contourner ce problème, utilisez [cors-anywhere](https://cors-anywhere.herokuapp.com/corsdemo) :

Rendez-vous sur ce lien et cliquez sur "Request temporary access to the demo server" avant de tester l'API en local.

### **Soleil**  
Le composant du soleil a été inspiré de ce [CodeSandbox](https://codesandbox.io/p/sandbox/glowing-sun-kpbyl).
